# 🏫 CampusPulse

**Smart Campus. Smarter Decisions.**

CampusPulse is a cloud-based smart campus monitoring and predictive analytics platform designed to improve campus operations through real-time monitoring, intelligent analytics, and AI-powered insights.

## Features

- 📊 Real-time Analytics
- 🏫 Smart Classroom Monitoring
- ⚡ Energy Consumption Dashboard
- 🔧 Maintenance Management
- 🤖 Predictive Analytics
- ☁️ Cloud Deployment (AWS)
